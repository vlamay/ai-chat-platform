# Быстрый старт фронтенда

## 🚀 Запустить dev сервер (30 сек)

```bash
cd ~/Documents/claude/ai-chat-platform/frontend
npm run dev
```

## 📱 Открыть в браузере

```
http://localhost:5173
```

## 🎯 Что вы увидите

1. **Login страница** ← Начало
2. Кнопка "Sign up" для регистрации
3. После регистрации → главная страница чата
4. Sidebar с чатами (пусто в начале)
5. Кнопка "+ New Chat"
6. Input для отправки сообщений

## ⚠️ Примечание

Frontend **работает без backend**. Вы сможете:
- ✅ Видеть UI
- ✅ Кликать кнопки
- ✅ Вводить текст
- ❌ Отправлять сообщения (будет ошибка - нет backend)

Для полного тестирования нужен backend!

## Следующий шаг

Когда будете готовы протестировать с backend:
1. Установите PostgreSQL ИЛИ развёрнете на Railway
2. Запустите backend (uvicorn)
3. Затем фронтенд подключится к backend

Ctrl+C
